# NotMiro 🚀

## Description
A collaborative canvas scrum tool, a clone of the popular Miro app.

## Features
- Real-time collaboration
- Email authentication
- Canvas tools: sticky notes, shapes, drawing, resizing/reshaping objects

## Tech Stack 🛠️    
- Next.js
- Convex
- Clerk
- Liveblocks
- TailwindCSS
- shadcn/ui
- sonner
- perfect-freehand
